# Sign in to deepsyte

deepsyte is a screenshot-as-a-service platform with an MCP server, REST API, CLI, and web dashboard — 46+ tools for screenshots, browser automation, CAPTCHA solving, email testing, and SEO/perf audits.

**What sign-in does**

- Opens your browser to `https://www.deepsyte.com/dashboard/keys`
- On success, the API key is stored securely in VS Code's secret storage
- Automatically writes `.vscode/mcp.json` so your agent (Copilot, Cursor, Continue, etc.) can call the MCP tools

**Already have a key?** Paste it directly when prompted.
