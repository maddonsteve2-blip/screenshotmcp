# Take your first screenshot

One command away. Type a URL, and the rendered screenshot opens in a WebView beside your editor.

**The inline panel includes**

- `Open` — open the hosted image in your browser
- `Copy URL` — copy the public screenshot URL to the clipboard
- `Rerun` — recapture the same URL without retyping it
- `View run` — deep-link to the run on the deepsyte dashboard

Behind the scenes, each screenshot is queued on Railway-hosted Playwright workers with stealth scripts and lazy-load handling.
