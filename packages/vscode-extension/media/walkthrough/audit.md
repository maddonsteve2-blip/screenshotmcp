# Run a UX audit

Audits combine a Chromium render with LLM-powered UX review. You get a structured report covering:

- Layout & hierarchy
- Readability & typography
- Accessibility signals
- Performance & Core Web Vitals
- SEO metadata

**The audit panel includes**

- A hero screenshot of the page
- Structured findings grouped by category
- `Re-audit` to re-run without retyping the URL
- `View run` to open the full run on the dashboard

Audits are great for quick sanity checks before shipping a landing page.
