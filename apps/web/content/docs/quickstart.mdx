## 1. Get Your API Key

Sign up at [screenshotsmcp.com](https://screenshotsmcp.com) and create an API key from your [dashboard](/dashboard/keys).

## 2. Add to Your IDE

### Cursor

Add to your Cursor MCP settings (`~/.cursor/mcp.json`):

```json
{
  "mcpServers": {
    "screenshotsmcp": {
      "url": "https://screenshotsmcp-api-production.up.railway.app/mcp/YOUR_API_KEY"
    }
  }
}
```

### Windsurf

Add to your Windsurf MCP config:

```json
{
  "mcpServers": {
    "screenshotsmcp": {
      "serverUrl": "https://screenshotsmcp-api-production.up.railway.app/mcp/YOUR_API_KEY"
    }
  }
}
```

### Claude Desktop

Add to `claude_desktop_config.json`:

```json
{
  "mcpServers": {
    "screenshotsmcp": {
      "url": "https://screenshotsmcp-api-production.up.railway.app/mcp/YOUR_API_KEY"
    }
  }
}
```

## 3. Start Using It

Ask your AI assistant:

```
Take a screenshot of https://stripe.com
```

That's it. Your AI can now see any website.
