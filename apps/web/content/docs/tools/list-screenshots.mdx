---
title: list_recent_screenshots
description: List the most recent screenshots taken with your API key.
---

## Parameters

| Parameter | Type | Required | Default | Description |
| --- | --- | --- | --- | --- |
| `limit` | number | No | 5 | Number of screenshots to return (1–20) |

## Example

```
Show me my last 10 screenshots
```

## Response

Returns an array of recent screenshots with URLs and metadata (timestamp, dimensions, format).
