---
title: browser_go_forward
description: Navigate forward in the browser history.
---

## Parameters

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `sessionId` | string | Yes | Session ID from `browser_navigate` |

## Example

```
Go forward to the next page
```

## Notes

- Equivalent to clicking the browser's forward button
- Only works if you've previously gone back
- Returns a screenshot of the page
