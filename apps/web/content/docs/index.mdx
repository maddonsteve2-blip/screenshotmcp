## What is ScreenshotsMCP?

ScreenshotsMCP is a **Model Context Protocol (MCP) server** that gives AI coding assistants the ability to see and interact with websites. It works with Claude, Cursor, Windsurf, VS Code, and any MCP-compatible client.

### 46+ Tools in One Server

| Category | What it does |
| --- | --- |
| **Screenshots** | Capture any URL — desktop, mobile, tablet, dark mode, full page, PDF |
| **Browser Sessions** | Navigate, click, fill forms, hover, scroll, press keys — like a real user |
| **Inspection** | Extract text, HTML, accessibility trees, run JavaScript |
| **Performance** | Core Web Vitals (LCP, FCP, CLS, TTFB), full network waterfall |
| **SEO** | Meta tags, Open Graph, Twitter Cards, headings, structured data, alt text audit |
| **Debugging** | Console logs, network errors, cookies, localStorage/sessionStorage |

### Quick Example

Once installed, just ask your AI assistant:

```
"Screenshot my site at https://myapp.com and tell me if the hero looks right"
```

Your AI will take a screenshot, see your UI, and give you actionable feedback — no setup, no code.
